Cybercity

AR roll-a-ball mobile game

2019

About

Inspired by cyberpunk aesthetic, Cybercity is an AR mobile game fused with neon lights and holograms. Mixing with a virtual environment with a real-life paper cube with QR code as controller, the core feature in this game is to control a ball rolling from the start point to the finish point, passing through different buildings. Using QR cube as tracker, players are able to tilt and rotate the cube to solve puzzles, reflecting the same movement in the scene. Various interactions such as getting on an elevator and taking a bullet train will be triggered along the journey.

This is an individual project.

Development
