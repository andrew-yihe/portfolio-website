24 Hours in Melbourne

Interactive online interface for tourists

2020

About

Designed for the tourists that would like to visit or are visiting the City of Melbourne, Australia, 24 Hours in Melbourne is a tool tailored for them to have an overview of the eateries, activities, sights and destinations at four time ranges of day and night.

Implemented using Mapbox customisable map system, approximately 200 points of interest are contained within this online visualisation interface.

This project is a collaborative effort. As team leader, I was responsible for all the idea sketching, low-fidelity prototyping, website development and presentation video editing and composing.

Development
Tools

Website development – HTML, CSS and JavaScript
Map editing – Mapbox Studio
Image assets design - Adobe Illustrator
Website prototyping – Adobe XD
